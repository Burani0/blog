My Experience in the Introduction to Software Development class The class was started on 19th of February and it is Conducted by Samson Kapeyi who is funny and talented teacher. basically the class was about introduction of html and css

I learnt about that I could share a platform with the best software engineers of all times which is GitHub. I am actually very happy, excited to be doing the things I thought geniuses could only do. This makes me feel special and very excited to be upgrading in my career

HTML

It was quite an overwhelming and explorative experience. The most interesting thing is that I could now access the Google code as I could now access the Google code as I was taught the inspection option.
